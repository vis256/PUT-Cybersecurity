Correlation means the relation between variables. Correlation analysis is used to "catch" whether there is a relation between two variables (properties, characteristics). What does correlation mean? Here it is necessary to understand the relation as a kind of similarity in the "behavior of two characteristics". When one feature, characteristic increases does the other also increase? Or does it decrease? Or does it not change at all?

When calculating the correlation relation between two variables, we get a value whose absolute value (in some correlation coefficients we get both negative and positive values) is called the strength of the correlation, the strength of the relation between two variables. Very often there is a doubt how to interpret the size of such a coefficient, does it mean a strong or weak relationship?

Classification according to J.Guilford:

$|\rho|$=0 - no correlation

$0.0<|\rho|0.1$ - weak correlation

$0.1<|\rho|0.3$ - weak correlation

$0.3<|\rho|0.5$ - average correlation

$0.5<|\rho|0.7$ - high correlation

$0.7<|\rho|0.9$ - very high correlation

$0.9<|\rho|<1.0$ - almost full correlation

$|\rho|=1$ - full correlation

Peterson's definition is the most commonly used:

According to this definition, the correlation coefficient $\rho(x,y)$ between two verialbles $x$ and $y$ is calculated as follows:

$$\rho(x,y)=\frac{cov(x,y)}{\sigma_x*\sigma_y}$$,

where:

  • $cov(x,y)$ is covariance between $x$ and $y$ $$cov(x,y)=E(xy)(E(x)E(y))$$
  • E(x) expected value of x
  • $\sigma_x$ standard deviation for varialble $x$

import numpy as np

def calculate_correlation(x, y):
    x = np.array(x)
    y = np.array(y)
    covariance = np.mean(x * y) - np.mean(x) * np.mean(y)
    correlation = covariance / (np.std(x) * np.std(y))
    return correlation

correlation = calculate_correlation(x, y)
print("Calculated correlation coefficient:", correlation)

Calculated correlation coefficient: 0.4297838676768133

#code
def generate_positive_correlation():
    X = np.random.rand(10)
    Y = X + np.random.normal(0, 0.1, 10)
    return X, Y

X, Y = generate_positive_correlation()
correlation = np.corrcoef(X, Y)[0,1]
print(correlation)

def generate_negative_correlation():
    X = np.random.rand(10)
    Y = -X + np.random.normal(0, 0.1, 10)
    return X, Y

X, Y = generate_negative_correlation()
correlation = np.corrcoef(X, Y)[0,1]
print(correlation)

def change_quarter_to_numeric_value(quarter_str):
    # quarter_str is 20XX-QX
    # convert it to 20XX * 10 + X
    year, quarter = quarter_str.split('-Q')
    return int(year) * 10 + int(quarter)
